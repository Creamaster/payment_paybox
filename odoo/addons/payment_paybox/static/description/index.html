Autorisez les paiements avec Paybox

Description

Ce module implémente les paiements via le prestataire de paiement Paybox. Il permet :

  • De payer via Paybox

Installation

Pour installer ce module, il suffit de se rendre dans le Back Office dans le menu "Application" et de rechercher "paybox" : Ce module va ajouter un nouveau mode de paiement accessible lors du réglement des commandes et factures.
La configuration du module se fait par le menu Comptabilité > Paiements > Intérmédiares de paiement > Paybox :

Les valeurs à modifier sont :

  • Numéro de site
  • Numéro de rang
  • Numéro d'identifiant
  • Clé d'authentification
  • Clé publique

Les 3 premières valeurs sont à récupérer sur le backoffice de Paybox.

La clé d'authentification se génére dans le backoffice de Paybox. Pour cela, une fois dans le backoffice, aller dans le menu "Information" et descendez tout en bas de la page. Vous aurez une interface ressemblant à ceci :
Il vous suffit maintenant d'écrire une phrase de passe (phrase, mot de passe ou n'importe quel texte). Il faut que votre phrase de passe respecte ces critères : la phrase doit contenir plus de 15 caractères et la force doit être supérieure à 90%. Une fois choisie, vous pouvez générer la clé en cliquant sur "Générer une clé".
Vous devez valider la clé en cliquant sur le bouton "Valider" en bas du cadre. Vous allez recevoir un email de confirmation contenant un lien pour activer la clé.
La clé générée se retrouvera dans le champ "Clé". Copiez et collez la dans le champ "Clé d'authentification" dans le backoffice d'Odoo.

La clé publique se récupère dans les manuels de Verifone. Rendez-vous sur ce lien. Cherchez dans les différents documents proposés "Clé Publique Paybox". Téléchargez le fichier, puis chargez le fichier dans le backoffice d'Odoo en cliquant sur le bouton "Chargez votre fichier".

Usage

Lors du paiement d'une commande, Paybox sera proposé comme moyen de paiement.
